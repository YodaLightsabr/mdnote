mdnote!

mdnote is a markdown-based online notepad focused on simplicity. Try creating a header with # or ##. Use ctrl (or cmd) b and i for bold and italic text. Create a list with - or * and use enter to add items. Write code with ` or a codeblock with ```. Remember to use shift+enter to continue the same element on a new line. More tools will be added soon.
